JavaScript Syntax Used in this program:


if - This thing just checks if the input satisfies the condition

let - This thing just basically assigns data types to the variables.

alert - This thing will basically pop up and show the user some info.

prompt - This thing will basically pop up and ask the user some info.

function - This thing basically just holds the main code and will get called once the button has been pressed.

JSON.parse - This thing basically just converts the user input *STRING* into *INTEGER* so the if statements can use it.
